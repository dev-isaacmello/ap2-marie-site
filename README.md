# AP2 | Mariê — Website Oficial (MPB/Pop)

Projeto acadêmico.

Objetivo: demonstrar uso de HTML5 semântico e Bootstrap 5 em contexto acadêmico.

Estrutura: três páginas estáticas — `index.html`, `sobre.html`, `agenda.html`.

Layout: navbar sticky-top, carousel com indicadores, modals e componentes interativos.

Componentes principais: carousel, modals (player/newsletter), tabela de agenda.

Responsividade: grid e utilitários Bootstrap para mobile, tablet e desktop.

Acessibilidade: marcação semântica, alt em imagens e contraste adequado.

Identidade visual: paleta escura com acentos em dourado (`#cca43b`).

Interatividade: formulários simulados e handlers JS para modals (client-side).

Mídia: embeds responsivos (Spotify iframe) e imagens com fallback remoto.

Deploy: implementação estática — basta servir os arquivos ou abrir `index.html`.

Testes recomendados: validar em DevTools e navegadores modernos (Chrome/Edge/Firefox).

Dependências externas: Bootstrap 5, Bootstrap Icons, Google Fonts.
Uso acadêmico: código fornecido para estudo, adaptável para exercícios de UI.

Estrutura de pastas: `css/`, `js/`, `img/`, páginas HTML na raiz.
Licença: material para avaliação acadêmica; não inclui ativos licenciados.
